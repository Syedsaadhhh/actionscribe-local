# Contributing to ActionScribe Hybrid

Thanks for your interest in improving ActionScribe Hybrid.

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests if available
5. Submit a pull request

## Contribution ideas

- Improve action extraction accuracy
- Add multilingual support (especially Urdu)
- Enhance offline model routing
- Add integrations (Notion, Todoist, WhatsApp workflows)
- Optimize low-RAM performance

## Code style

- Keep functions modular
- Add docstrings
- Avoid heavy dependencies unless optional

Small improvements are welcome. Practical improvements are loved.
